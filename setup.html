DOWNLOADING STB EMULATOR

Installing through Kodi

  1. Open Kodi.
  2. Scroll to Programs on the Kodi menu bar.
  3. Open the APK Downloader.
  4. Select Android.
  5. Go to JayTech Apps.
  6. Select the latest version of the STB Emulator.
  7. Select Download.
  8. When the download is complete, select Install.
  9. When the installation is complete, select Done.
  10. Exit Kodi and return to your Android box's home screen.

NOTE: This installation option only works on select Android devices. If you are unable to install the STB Emulator using this method, select the installation instructions for Android.

Installing on Android

  1. Download the STB Emulator installation file from http://download.iptvfrog.com
  2. Transfer this file to your Android device, if necessary (usually via USB).
  3. Open the APK file.
  4. Select Install.
  5. When the installation is complete, select Done.

Installing on Amazon Fire TV

  1. Go to the Amazon Fire TV settings.
  2. Select Device.
  3. Click Developer Options.
  4. Ensure that ADB Debugging is On.
  5. Ensure that Apps From Unknown Sources is On.
  6. Open the Downloader app.
  7. Select Browser on the sidebar.
  8. Go to http://download.iptvfrog.com
  9. When the file is done downloading, select Install.
  10. When the app is done installing, select Done.

CONFIGURING STB EMULATOR

  1. Open the STB Emulator.
  2. After it opens, press the Configure Application button OR press the menu key on your remote (looks like a list) and select Settings from the drop down menu that appears.
  3. From the sidebar, select Profiles.
  4. If there are no profiles listed on your program, press the Add Profile button. If there is a profile listed, delete it.
  5. In the profile settings, select Profile Name.
  6. Press the return key on your remote to hide the onscreen keyboard.
  7. Press and hold the right arrow key on your remote to move the text cursor to the end of the name.
  8. Press the OK button on your remote to bring back the onscreen keyboard.
  9. Erase the existing name by pressing and holding the backspace button on the onscreen keyboard on your remote.
  10. Now type Blue IPTV Frog into the text box.
  11. Press the return key on the remote to hide the onscreen keyboard.
  12. Press Ok on the text box.
  13. From the portal settings, select the Portal URL.
  14. Press the return key on your remote to hide the onscreen keyboard.
  15. Press and hold the right arrow key on your remote to move the text cursor to the end of the URL.
  16. Press the OK button on your remote to bring back the onscreen keyboard.
  17. Erase the existing URL by pressing and holding the backspace button on the onscreen keyboard on your remote.
  18. Now type http://blue.iptvfrog.com into the text box.
  19. Press the return key on the remote to hide the onscreen keyboard.
  20. Press Ok on the text box.
  21. Press the return key on the remote to return to the profile settings screen.
  22. Now select the STB Configuration settings.
  23. Open the MAC-address setting.
  24. Press the return key on your remote to hide the onscreen keyboard.
  25. Press and hold the right arrow key on your remote to move the text cursor to the end of the MAC address.
  26. Press the OK button on your remote to bring back the onscreen keyboard.
  27. Erase the existing MAC address by pressing and holding the backspace button on the onscreen keyboard on your remote.
  28. Now type [Your MAC Address] into the text box.
  29. Press the return key on the remote to hide the onscreen keyboard.
  30. Press Ok on the text box.
  31. Repeatedly press the return key on the remote until you return to the STB Emulator main menu.
  32. Press the return key on the remote until you return to the STB Emulator profiles menu.
  33. Press the Add Profile button.
  34. In the profile settings, select Profile Name.
  35. Press the return key on your remote to hide the onscreen keyboard.
  36. Press and hold the right arrow key on your remote to move the text cursor to the end of the name.
  37. Press the OK button on your remote to bring back the onscreen keyboard.
  38. Erase the existing name by pressing and holding the backspace button on the onscreen keyboard on your remote.
  39. Now type Yellow IPTV Frog into the text box.
  40. Press the return key on the remote to hide the onscreen keyboard.
  41. Press Ok on the text box.
  42. From the portal settings, select the Portal URL.
  43. Press the return key on your remote to hide the onscreen keyboard.
  44. Press and hold the right arrow key on your remote to move the text cursor to the end of the URL.
  45. Press the OK button on your remote to bring back the onscreen keyboard.
  46. Erase the existing URL by pressing and holding the backspace button on the onscreen keyboard on your remote.
  47. Now type http://yellow.iptvfrog.com into the text box.
  48. Press the return key on the remote to hide the onscreen keyboard.
  49. Press Ok on the text box.
  50. Repeatedly press the return key on the remote until you return to the STB Emulator main menu.
  51. Press the menu key on your remote.
  52. Select Exit from the drop-down menu to close the STB Emulator.
  53. Reopen the STB Emulator and your TV menu should now display.

NOTE: If you're account has not been activated yet, you will be asked for login information. This is normal and cannot be bypassed until your account has been activated on our end. This can take up to two business days, after which point your IPTV will automatically start working.
